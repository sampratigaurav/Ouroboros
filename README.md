# Project Ouroboros

Real-time multiplayer competitive snake arena.

## Features

- Multiplayer battles
- Solo mode with AI bots
- Dash ability
- Trap system
- Shrinking arena
- Wormholes
- Leaderboard

## Tech Stack

Frontend:
HTML, CSS, JavaScript, Canvas

Backend:
Node.js, Express, Socket.IO

## How to run

npm install
npm start
